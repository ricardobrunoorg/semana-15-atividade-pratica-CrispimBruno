Sobre o Mesa Aberta

O Mesa Aberta é um catálogo de jogos de tabuleiro e RPG de mesa, criado como Trabalho Interdisciplinar 1 do curso de Engenharia de Software / Desenvolvimento de Interfaces Web da PUC Minas.

O projeto integra um módulo de login (autenticação via sessionStorage e API JSON Server) com uma funcionalidade de personalização: usuários autenticados podem marcar jogos como favoritos, persistidos por conta em localStorage.
